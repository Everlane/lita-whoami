# lita-whoami

This handler allows you to tell Lita who people are

## Installation

Add lita-whoami to your Lita instance's Gemfile:

``` ruby
gem "lita-whoami"
```

## Usage

```
lita taylor is a bad programmer
```

```
lita who is taylor
```

```
lita taylor isn't a bad programmer
```

```
lita I don't know who anyone is
```
